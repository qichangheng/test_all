## 什么单元测试？？

单元：函数或者是类
单元测试：测试类或者函数

## 为什么要做单元测试？？
好处：投入小，收益大。能够精准的，更早的发现问题。

## 单元测试和我有什么关系？？
python 很难测试 java 的单元。
关键是单元测试一般是开发或者测试开发做的。
